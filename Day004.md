# 객체지향 프로그래밍

## 객체지향 프로그래밍

> 객체지향 프로그래밍(OOP, Object Oriented Programming)

- 객체 : 사물과 같이 유형적인 것과 개념이나 논리와 같은 무형적인 것들
- 지향 : 작정하거나 지정한 방향으로 나아감
- 객체 모델링 : 현실세계의 객체를 SW 객체로 설계하는 것

> 객체 지향 프로그래밍 특징 (A PIE)

  - Abstraction(추상화)
  - Polymorphism(다형성)
  - Inheritance(상속)
  - Encapsulation(캡슐화)
